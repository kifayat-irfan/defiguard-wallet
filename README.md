# 🔐 DeFiGuard Wallet

A Web3 wallet interface built with Web3.js for the DeFiGuard technical assessment.

## ✅ Features
- MetaMask wallet connection
- Live ETH balance and address display
- Send ETH functionality
- Sepolia Testnet support

## 🛠️ Prerequisites
- Node.js v18+
- Git
- MetaMask browser extension

## 🚀 Setup & Run
git clone https://github.com/kifayat-irfan/defiguard-wallet.git
cd defiguard-wallet
npm install
npm install web3
npx live-server public

## 🧪 Testing
1. Open http://127.0.0.1:8080
2. Switch MetaMask to Sepolia Testnet
3. Click Connect MetaMask
4. Address and balance will display
5. Use Send ETH form to transfer tokens
